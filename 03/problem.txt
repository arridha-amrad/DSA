Question : Remove duplicates from sorted array

Given an ineteger array 'nums' sorted in non-decreasing order.
Remove the duplicates in-place such that unique element appears only once.
The relative order of the array should be kept the same.
Then return the number of unique elements inside the array

Example:
  Input: [1,1,2]                 ---->>> Output: 2
  Input: [0,0,1,1,1,2,2,3,3,4]   ---->>> Output: 5
